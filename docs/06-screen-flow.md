## 標準日次運用

```mermaid
sequenceDiagram
  participant 管 as 管理者
  participant 現 as 現場LINE
  participant 系 as システム
  participant 社 as 社長
  現->>系: 作業開始（工程）
  現->>系: 作業終了
  系->>系: 人別・工程別・人別工程時間を算出
  管->>系: 前日の入荷・個人加工・製造・納品を入力
  系->>系: 個人実績から工程実績を集計
  系->>系: 在庫元帳とDashboard表示値を原本から再計算
  管->>系: 部門別に確認済みへ変更
  社->>系: 朝、前日・れんこんを表示
  系-->>社: 判断用トップ（加工投入・完成・工程歩留まり・出荷売上・完成品在庫）
  社->>系: 工程／人員／採算／在庫の詳細を選択
  系-->>社: 原因確認用詳細
  社->>系: 青果タブへ切替
  系-->>社: 同一定義の青果実績
  社->>系: 週次・月次へ切替
  系-->>社: 同じ画面で期間集計
```

締め済み訂正は、管理者が理由を入力して日次を再開し、修正後に再確認します。社長画面は常に現在状態と最終確認日時を表示します。
