# ⑤ 画面一覧

## Phase2A追加画面

- LINE利用者管理（管理者PIN）: 登録状態、登録コード発行、解除
- 作業履歴・訂正（管理者PIN）: 日付・担当者・部門・工程検索、理由付き訂正
- 会社設定（管理者PIN）: 終業時刻、猶予、休憩方式、LINE/通知、固定休憩、LINEメニュー編集

## Phase1

社長ロールにはS02～S06だけを表示し、更新APIも許可しません。Dashboardトップは判断用、各詳細画面は原因確認用です。人別歩留まりの表示実体はS04に一元化し、S03から必要に応じて同じ詳細へ遷移します。

Dashboardおよび集計画面に入力・直接編集機能を置きません。

## 入力画面の最小必須項目

|画面|必須|自動・任意|
|---|---|---|
|入荷|日付、部門、品目、数量、単位、ロット|取引先、金額、期限、メモ|
|個人加工|作業日、部門、工程、使用材料ロット、使用量g、破棄g、出来高g、担当者|人別歩留まり自動、時間はLINE参照、備考|
|製造|日付、部門、ロット、使用品と量、完成商品、完成重量、完成数量・単位|フレーバー・包材消費、不良・廃棄、時間、メモ|
|納品・売上|納品日、部門、納品先、商品、数量、売掛／現金|売掛時の入金予定日、単価・売上・直接原価・粗利自動、物理出荷日、メモ|
|棚卸|日付、部門、品目、実数|帳簿数・差異自動、理由|
