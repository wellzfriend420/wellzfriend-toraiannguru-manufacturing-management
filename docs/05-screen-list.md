# ⑤ 画面一覧

## Phase2A追加画面

- LINE利用者管理（管理者PIN）: 登録状態、登録コード発行、解除
- 作業履歴・訂正（管理者PIN）: 日付・担当者・部門・工程検索、理由付き訂正
- 会社設定（管理者PIN）: 終業時刻、猶予、休憩方式、LINE/通知、固定休憩、LINEメニュー編集

# モバイルUI共通方針（2026-07-22承認）

- スマホを主利用端末とし、横スクロールを禁止する。
- 主要導線は「ホーム・入力・在庫・メニュー」の下部ナビゲーションとする。
- ダッシュボードと入力画面は同時表示しない。
- マスタ管理は折り返しタブで種別を切り替え、一覧と追加・編集を別画面として扱う。
- 一覧は横長表ではなく縦型カード、入力は1画面1業務を原則とする。
- 人物写真・人物イラスト・過度に可愛い装飾は使わず、アイボリー・深緑・セージを基調とする。
- ボタン・入力欄の操作領域は原則44px以上とする。
